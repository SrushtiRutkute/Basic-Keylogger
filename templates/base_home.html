About this demo

This site demonstrates an ethical, consented keystroke capture tool. Nothing is recorded unless a user checks the consent box and starts logging. Logs remain on the server only if the user chooses to upload them and an admin token is provided.

Important: Do not use this to record other people without their explicit informed consent. Covert logging is illegal and unethical.
